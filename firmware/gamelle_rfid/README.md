# Gamelle RFID pour chat

Ouverture de la trappe par tag RFID (collier). Option 125 kHz (RDM6300) ou 134,2 kHz FDX-B (puce implantée).

**Matériel** : lecteur RFID (RDM6300 ou FDX-B), Arduino ou ESP32, servo MG90S, LEDs, buzzer. Voir [guide_domotique_complet.md](../../guide_domotique_complet.md) section 4 pour le câblage et le code.
