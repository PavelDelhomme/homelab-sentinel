# Prise connectée ESP32

Prise DIY : commande on/off + mesure tension/courant/puissance, envoi des données en MQTT.

**Matériel** : ESP32 WROOM-32D, relais 5V, ACS712 (courant), ZMPT101B (tension), HI-Link 5V, prise 230V. Voir [guide_domotique_complet.md](../../guide_domotique_complet.md) section 3 pour le schéma, le câblage et le code Arduino/ESP32 complet.
