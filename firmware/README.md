# Firmware Arduino / ESP32

Emplacements pour les sketches du projet Homelab Sentinel. Le code détaillé (prises connectées, gamelle RFID) est dans le **guide à la racine** : [../guide_domotique_complet.md](../guide_domotique_complet.md).

- **prises_esp32** — Prise connectée avec mesure de consommation (ESP32, ACS712, ZMPT101B, relais). Voir guide section 3.
- **gamelle_rfid** — Gamelle chat avec lecteur RFID (RDM6300 ou FDX-B) et servo. Voir guide section 4.

Rien à compiler tant que l’Arduino / l’ESP32 n’est pas connecté et le matériel acquis.
